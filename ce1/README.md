## Week 1: Basic Data Exploration and Cleaning
This repository contains the first week's assignment focusing on utilizing Pandas for handling real-world ecommerce data.

### Project Overview
The goal of this project was to take a raw dataset from Kaggle (shopping_data) and perform essential data engineering tasks including cleaning, transformation, and feature engineering.

### Key Tasks Completed:
1. **Data Profiling:** Explored a dataset of 1,000 products with 24 features.
2. **Null Management:** Handled missing values using a hybrid strategy of imputation for descriptive data and numerical defaults for discounts.
3. **Data Cleaning:** Developed a robust cleaning pipeline to strip literal quotes, currency symbols, and commas from string-based price columns.
4. **Feature Engineering:** Created a derived `total_amount` column using numeric transformation and multiplication logic.
5. **Data Export:** Successfully generated a cleaned CSV for downstream usage.
